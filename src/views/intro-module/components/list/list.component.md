# 列表

## 普通列表

```html
<div class="list">
    <a href="javascript:;" class="list-item">列表项1</a>
    <a href="javascript:;" class="list-item active">列表项2</a>
    <a href="javascript:;" class="list-item">列表项3</a>
    <a href="javascript:;" class="list-item">列表项4</a>
    <a href="javascript:;" class="list-item">列表项5</a>
</div>
```
