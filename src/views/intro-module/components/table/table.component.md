# 表格

## 表格内辅助样式

```html
<table class="table">
    <thead>
        <tr>
            <th>表头1</th>
            <th>表头2</th>
            <th>表头3</th>
            <th>表头4</th>
            <th>表头5</th>
        </tr>
    </thead>
    <tbody>
        <tr class="primary">
            <td>内容1</td>
            <td>内容2</td>
            <td>内容3</td>
            <td>内容4</td>
            <td>内容5</td>
        </tr>
        <tr class="info">
            <td>内容1</td>
            <td>内容2</td>
            <td>内容3</td>
            <td>内容4</td>
            <td>内容5</td>
        </tr>
        <tr class="success">
            <td>内容1</td>
            <td>内容2</td>
            <td>内容3</td>
            <td>内容4</td>
            <td>内容5</td>
        </tr>
        <tr class="warning">
            <td>内容1</td>
            <td>内容2</td>
            <td>内容3</td>
            <td>内容4</td>
            <td>内容5</td>
        </tr>
        <tr class="danger">
            <td>内容1</td>
            <td>内容2</td>
            <td>内容3</td>
            <td>内容4</td>
            <td>内容5</td>
        </tr>
        <tr>
            <td class="primary">内容1</td>
            <td class="info">内容2</td>
            <td class="success">内容3</td>
            <td class="warning">内容4</td>
            <td class="danger">内容5</td>
        </tr>
    </tbody>
</table>
```
