# 单选框

### @Input()

| 输入属性名 | 输入类型  | 说明    |
| --        | --        | --        |
| `value`      | `string`   |  |
| `name`      | `string`   |  |
| `disabled`      | `boolean`   |  |
| `readonly`      | `boolean`   |  |
| `checked`      | `boolean`   |  |
| `checkedIcon`      | `string`   | 在选中时候的icon |
| `uncheckedIcon`      | `string`   |  在未选中时候的icon |

### @Output()
| 输出属性名 | 输出值（$event）  | 说明    |
| --        | --        | --        |
| `change` | `string`   | 当前组件的value |
