# 区间选择器

### @Input()

| 输入属性名 | 输入类型  | 说明    |
| --        | --        | --        |
| `value`      | `string`   |  |
| `name`      | `string`   |  |
| `disabled`      | `boolean`   |  |
| `readonly`      | `boolean`   |  |
| `max`      | `number`   | 最大值 |
| `min`      | `number`   |  最小值 |
| `step`      | `number`   |  步长 |

### @Output()
| 输出属性名 | 输出值（$event）  | 说明    |
| --        | --        | --        |
| `change` | `number`   | 当前组件的value |
